# Peculiar Charm
A Gallery of the Unique and Odd. User can login to view, edit, contribute to or delete pieces in the collection.

## Technoligies Used
- CSS
- HTML
- JavaScript
 -Node.JS
- MongoDB
- Mongoose
- Express
- Bootstrap
